Drowsiness Detection System

Role: Team Lead · Computer Vision Engineer

Problem

Driver fatigue is a major risk factor and needs reliable real-time detection for preventive alerting.

Approach

  • Implemented computer-vision based eye state and drowsiness monitoring.
  • Integrated Raspberry Pi hardware for portable deployment.
  • Designed alert triggering logic for practical field scenarios.

Impact

  • Demonstrated end-to-end real-time fatigue detection prototype.
  • Validated CV + edge hardware feasibility for safety use cases.

Stack

Python, OpenCV, Raspberry Pi, real-time image processing.
